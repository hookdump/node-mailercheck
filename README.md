# Usage

Check a single email and get real-time response

```js
const MailerCheck = require('../index')

const mailercheck = new MailerCheck(
    {
        api_key: "api_key"
    }
)

mailercheck.checkEmail("test@mailercheck.com")
    .then((response) => {
        console.log(response)
    })
    .catch(err => {
        console.log(err)
    })
```

Response

```js
{ 
    message: 'valid', 
    status: 200 
}
```
